In short,

reveal.js presentations are html.

Ruby is good at producing html.

Let's produce reveal.js presentations with minimal work.

mission statement

reveal.js is Great!

reveal.js is great choice for creating slides. Check out the live demo for examples of all the things it can do.

Smaller, Simpler, Better

reveal-ck allows you to reduce a presentation down to a single file.

You can focus on your content and message when you manage a small number of files that generate a larger number.

Use Tools that you Know

reveal-ck presentations are made of text. You probably have a favorite way of editing text and tracking changes in text.

You can use the same tools for presentations that you use for text.

Browsers: Good for Slides

What makes for a good presentation? Beautiful images? Well-chosen typography? Overall styling? Music? Video? The modern browser is powerful and ubiquitous.

Picking the browser as a slide runtime is a good choice.

Quick Start, Advanced Features

reveal-ck supports Markdown, which lets you making your first presentation with ease.

Yet, Markdown itself is limited when it comes to creating very specific html. And so, reveal-ck supports an abundance of other html generation languages if you need them.
