It's Just Html

Ruby has many tools that generate html and has famous static site generators.

reveal.js presentations are static html.

reveal-ck is a static slide generator.

Start with Markdown

reveal-ck slightly tweaks Markdown so that it produces html that reveal.js recognizes.

Create slides even if you don't understand how reveal.js works.

Use Other Languages (if needed)

If you want to take the time to understand how reveal.js works, you will need finer control over the html in your slides.

Author slides in the language of your choice.

Generate Slides

There's support for a Ruby internal DSL that can generate slides.

Want to create a presentation with a thousand slides based on a flickr search? Want it to automatically change slides every 10 seconds and loop?

Want a presentation that plays music with each slide, and the music it plays comes from searching your file system?

Unlock a new category of possibilities with generated slides.
